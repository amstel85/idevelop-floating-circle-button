A simple Floating Circle Button for your WordPress site.

== Description ==
This plugin adds a customizable Floating Circle Button to your WordPress website.
It allows visitors to easily initiate a new conversation directly in the WhatsApp application.
You can configure the button's appearance, position, and the pre-filled message for the conversation.
This version includes general improvements and bug fixes.

== Installation ==
1.  **Automatic Installation:**
    *   Go to your WordPress dashboard, navigate to 'Plugins' -> 'Add New'.
    *   Search for 'iDevelop Floating Circle Button' and click 'Install Now'.
    *   Activate the plugin through the 'Plugins' menu.

2.  **Manual Installation (via Zip Upload):**
    *   Download the plugin zip file from the GitHub releases page.
    *   Go to your WordPress dashboard, navigate to 'Plugins' -> 'Add New' -> 'Upload Plugin'.
    *   Choose the downloaded zip file and click 'Install Now'.
    *   Activate the plugin through the 'Plugins' menu.

3.  **Manual Installation (via FTP):**
    *   Upload the `idevelop-floating-circle-button` folder to the `/wp-content/plugins/` directory via FTP.
    *   Activate the plugin through the 'Plugins' menu in WordPress.

4.  **Configuration:**
    *   Configure the plugin settings under 'Settings' -> 'Floating Circle Button'.

== Customizable Options ==
Beyond appearance, position, and pre-filled message, the plugin can be enhanced with the following customizable options:

*   **Display Conditions & Targeting:**
    *   Control visibility based on device type (desktop, mobile, tablet).
    *   Introduce a delay before the button appears.
    *   Make the button appear after scrolling a certain percentage down the page.

*   **Behavioral & Interaction Options:**
    *   Configure whether the chat opens in the same tab or a new one.

*   **Advanced Pre-filled Message Customization:**
    *   Allow dynamic message content (e.g., current page URL, page title, custom field values).

*   **User Interface Enhancements:**
    *   Add a customizable text label next to the icon (e.g., "Chat with us").
    *   Apply subtle animation effects (e.g., pulse, fade-in).
    *   Display a short call-to-action text on hover.
